Complex CSS Shapes

Web designers have long been forced to create within the constraints of the block. Since most daring ventures into non-rectangular layout using CSS and HTML end in frustration, most content on the web is still trapped in simple boxes. Border-radius and creative uses of CSS transform properties only create the illusion of shapes; they don't actually affect the content layout as expected. A circle carved with border-radius is still a rectangle in disguise.

All this is about to change with the implementation of CSS Shapes, a new CSS feature which allows web designers to wrap content inside and around custom shapes. Shapes are defined in simple, practical and reusable ways, and, unlike most CSS hacks that just simulate shapes, they can affect the entire layout of a web page.
